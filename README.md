# ⚛️ Tabla Periódica - Juego Educativo

Juego interactivo para aprender la tabla periódica jugando. 100% responsive para móvil, tablet y escritorio.

## 🎮 3 Modos de Juego

📚 **Aprende**: Tarjetas interactivas con hechos de cada elemento  
🎯 **Quiz**: Trivia con 10 preguntas - 10 puntos por respuesta correcta  
🎲 **Clasifica**: Arrastra elementos a sus categorías - 5 puntos por clasificación  

## 🚀 Desplegar en Vercel

### Paso 1: Crear repositorio en GitHub
```bash
# En c:\Users\jalva\OneDrive\Documentos\Programas\periodic-table
git init
git add .
git commit -m "Tabla Periódica Game"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/tabla-periodica.git
git push -u origin main
```

### Paso 2: Conectar a Vercel
1. Ve a [vercel.com](https://vercel.com)
2. Inicia sesión / Crea cuenta
3. Click "New Project"
4. Selecciona tu repo `tabla-periodica`
5. Click "Deploy" ✅

**¡Listo! Tu app estará en:** `https://tabla-periodica.vercel.app`

## 🎮 Jugar Localmente

```bash
npm start
# Abre http://localhost:3000
```
